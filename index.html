jquery-autocomplete

jQuery Autocomplete plugin

A simple but powerful autocomplete plugin for jQuery.

Usage

See the examples page.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/reinh/jquery-autocomplete

get the source code on GitHub : reinh/jquery-autocomplete
